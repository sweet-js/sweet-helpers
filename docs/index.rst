Sweet.js Helpers
=========================================

The ``@sweet-js/helpers`` module is a collection of utilities for matching syntax in macros.

The utilities are broken down into two main areas:

* :doc:`parser-combinators` -- A high-level API for building expressive parsers that match syntax.
* :doc:`helpers` -- A low-level API for building high-level APIs.
